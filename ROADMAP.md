# Roadmap

## Near Term

- Improve the company relationship graph.
- Expand upstream and downstream supply-chain tracing.
- Add more verified company descriptions.
- Improve the intelligence radar and source-review workflow.
- Strengthen mobile navigation and reading experience.

## Mid Term

- Add a more complete data-source library.
- Add source confidence labels and evidence status.
- Add member-level saved subscriptions.
- Improve localization quality across English, Chinese, Japanese, and Korean.
- Add more structured industry reports and source citations.

## Long Term

- Build a deeper AI hardware supply-chain knowledge base.
- Support customer-supplier chain tracing.
- Add more public market and private company coverage.
- Build a repeatable research workflow for new bottlenecks.

## Not Planned For This Public Repository

- Open sourcing production frontend code.
- Open sourcing backend code.
- Publishing crawler logic.
- Publishing paid datasets.
- Publishing API credentials or deployment configuration.
